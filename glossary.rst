*********
Глоссарий
*********

.. glossary::

  Бандл
    (Bundle) Крупный бизнес-логический модуль в CordJS. Название украдено из PHP-фреймворка Symfony. Подробнее о
    бандлах в :doc:`соответствующей главе <general/bundles>`.

  Промис
    (Promise) Одна из самых известных программных абстракций для асинхронных вычислений. Объект, содержащий результат
    асинхронного вычисления, который будет доступен в какой-то момент в будущем. В CordJS используется собственная
    реалзиация под названием :doc:`Future <utils/future>`.
